Detailed Description

Represents a singleton callback function for handling key events in a GLFW window.

The GlfwWindowContentScaleChangedCallbackFunction class provides a singleton instance of a callback function that can be used to handle content scale changed events in a GLFW window. It ensures that only one instance of the callback function exists at a time.
