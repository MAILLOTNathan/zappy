\doxysubsection{Detailed Description}
Represents a map in the game. 

The \doxylink{classOnyx_1_1Map}{Map} class is a derived class of the \doxylink{classOnyx_1_1Entity}{Onyx\+::\+Entity} class and represents a map in the game. It contains a collection of floor objects and provides functionality to update the map using a shader.
