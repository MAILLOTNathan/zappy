\doxysubsection{Detailed Description}
Represents a singleton callback function for handling key events in a GLFW window. 

The \doxylink{classEGE_1_1GlfwWindowFramebufferResizedCallbackFunction}{Glfw\+Window\+Framebuffer\+Resized\+Callback\+Function} class provides a singleton instance of a callback function that can be used to handle framebuffer resized events in a GLFW window. It ensures that only one instance of the callback function exists at a time.
