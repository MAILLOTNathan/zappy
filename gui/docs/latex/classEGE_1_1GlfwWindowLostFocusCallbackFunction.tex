\doxysubsection{Detailed Description}
Represents a callback function for the GLFW window lost focus event. 

This class provides a mechanism to register a callback function that will be called when a GLFW window loses focus. Only one instance of this class can exist at a time, and it follows the singleton design pattern. The callback function must be invocable with parameters GLFWwindow\texorpdfstring{$\ast$}{*} and int.
